# Sora Unity SDK サンプル集

このディレクトリ以下には、 [Sora Unity SDK](../) を利用したサンプルアプリを掲載しています。実際の利用シーンに即したサンプルをご用意しておりますので、目的に応じた Sora Unity SDK の使い方を簡単に学ぶことができます。

## 対応 Unity バージョン

- Unity 6000.0 (LTS)

## 対応 Sora バージョン

- WebRTC SFU Sora 2025.1.0 以降

## 対応 プラットフォーム

- Windows 10 22H2 x86_64 以降
- macOS 13.4.1 M1 以降
- Android 7 以降
- iOS 13 以降
- Ubuntu 22.04 x86_64
- Ubuntu 24.04 x86_64

## Sora Unity SDK サンプル集を使ってみる

Sora Unity SDK サンプル集の使い方についてはドキュメントのチュートリアルを参照してください。

[Sora Unity SDK ドキュメントのチュートリアル](https://sora-unity-sdk.shiguredo.jp/samples_tutorial)
